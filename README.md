# 配置
打開`.env`
```
API_KEY=如果你使用elevenlabs之類的服務在填
TOKEN=discord機器人token
SERVICE="google"
```
- SERVICE可填入 :
    - [google](https://translate.google.com.tw/) 
    - [elevenlabs](https://elevenlabs.io/)
